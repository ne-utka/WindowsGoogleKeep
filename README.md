Desktop-приложение для Google Keep на Electron с сохранением сессии и отдельной верхней панелью управления окном.

### Возможности

- Постоянная сессия входа (`persist:keep`)
- Открытие только разрешённых доменов:
  - `https://keep.google.com/*`
  - `https://accounts.google.com/*`
- Остальные внешние ссылки открываются в браузере
- Кастомная верхняя панель окна:
  - свернуть
  - развернуть/восстановить
  - закрыть
- Настраиваемая иконка приложения

### Запуск

```bash
npm install
npm start
```

## English

An Electron desktop wrapper for Google Keep with persistent session storage and a custom top window control bar.

### Features

- Persistent login session (`persist:keep`)
- Only whitelisted domains are allowed in-app:
  - `https://keep.google.com/*`
  - `https://accounts.google.com/*`
- Other external links are opened in the default browser
- Custom top window controls:
  - minimize
  - maximize/restore
  - close
- Custom app icon support

### Run

```bash
npm install
npm start
```
